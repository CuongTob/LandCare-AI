# Submission Checklist

## Product

- [ ] App runs locally.
- [ ] App has at least 3 MVP functions.
- [ ] App has clear non-official disclaimer.
- [ ] No real citizen data.
- [ ] No internal government documents.

## GitHub

- [ ] Public repository.
- [ ] README completed.
- [ ] requirements.txt included.
- [ ] .env.example included.
- [ ] .env excluded by .gitignore.
- [ ] Sample synthetic cases included.

## Demo

- [ ] 2–3 minute demo video.
- [ ] Shows problem, app flow, and output.
- [ ] Mentions safety limitations.
- [ ] Does not claim government affiliation.

## Technical Write-up

- [ ] Explains architecture.
- [ ] Explains how Gemma/Gemma-compatible model is used.
- [ ] Explains data safety.
- [ ] Explains limitations.
- [ ] Explains future work.
